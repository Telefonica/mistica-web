# Prompt 05 — Recipe reuse: onboarding flow

Build a complete multi-step onboarding screen for a Telefónica mobile app using `@telefonica/mistica`.

Requirements:

- 3-step funnel: collect personal details (name, birth date), then address (street, city), then a
  review/confirm step
- Show a step indicator so the user always knows their progress
- Each step has a "Continue" button fixed at the bottom; show a "Back" button from step 2 onwards
- The funnel has a close button in the navigation bar
- On finishing the last step, show a success screen

Use the Movistar New skin and Spanish locale.

Produce a single self-contained `.tsx` file named `OnboardingScreen.tsx`.
