The component is fully self-contained. Key design decisions:

- **Navbar**: fixed-position with fade-to-transparent gradient, active section underline indicator, search icon (`IconSearchRegular`), and `Avatar` for the profile
- **Hero**: full-viewport-height with `position: absolute` image + gradient scrim; buttons are wrapped in `ThemeVariant variant="inverse"` so mistica renders them in high-contrast inverse mode against the dark background
- **Content rows**: `HorizontalScroll` with inline `CoverCard` items (160px wide, 2:3 ratio); tags use mistica's `Tag` component via the `headline` prop
- **My List**: toggled via `activeSection` state; renders mistica's `EmptyState` with image, description, and a CTA button
- **Theme showcase**: a `ThemeVariant variant="media"` dark row followed by a `ThemeVariant variant="brand"` gradient CTA banner — demonstrating mistica's theming system across sections
- All colors reference `skinVars.colors.*` tokens (consumer-facing API) rather than hardcoded values, with the exception of the intentional cinematic chrome (navbar gradient, hero scrim) which uses rgba blacks by design
