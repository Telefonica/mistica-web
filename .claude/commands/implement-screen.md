# /implement-screen

Load the `mistica-react` skill, then build a complex UI screen.

## Instructions

1. Read `skills/mistica-react/SKILL.md` — the "Component mapping for complex screens" section has the UI zone
   → component table and page structure template
2. Read `doc/llms.md`, `doc/patterns.md`, `doc/components.md`, and `doc/layout.md` per the skill's Step 1
   instructions
3. Follow the skill's hard constraints exactly (no hardcoded colors, no raw divs, namespaced hooks, etc.)
4. Output the complete file as a labelled code block

The user will provide: screen description and requirements.
