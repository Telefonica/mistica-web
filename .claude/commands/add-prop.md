# /add-prop

Load the `mistica-contributor` skill, then add a prop to an existing component.

## Instructions

1. Read `skills/mistica-contributor/SKILL.md` — specifically "Task: add a prop to an existing component"
2. Read the current files for the named component before making any changes
3. Update all 4 locations (tsx, css.ts, story, playroom snippet) — never skip any
4. Run `yarn ts-check` and report any errors

The user will provide: component name, prop name, prop type, and behaviour description.
